# Enso Intelligence Website

One-page marketing site for **enso-intelligence.com**.

## Tech Stack
- React + TypeScript (Vite)
- Tailwind CSS
- Framer Motion
- shadcn/ui primitives

## Local Development
```bash
npm install
npm run dev
```

## Production Build
```bash
npm run build
npm run preview
```

## Page Sections
- Hero
- Mission (Why We Exist)
- Products
- Why Enso
- Team
- Contact
- Footer
